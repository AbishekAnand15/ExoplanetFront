FAQ

What is a TIC ID?

A TIC (TESS Input Catalog) ID uniquely identifies a star observed by NASA‚Äôs TESS mission.

Does this confirm exoplanets?

No. This tool identifies candidate transit signals. Scientific confirmation requires follow-up observations.

Why does analysis sometimes take time?

The backend fetches real astronomical data and may take longer if the server is waking up or the dataset is large.
